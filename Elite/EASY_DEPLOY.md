## Option 2: GitHub Pages (Free Forever)

### Quick Setup:

1. **Create GitHub account** (if you don't have one): https://github.com/signup

2. **Create new repository:**
   - Go to: https://github.com/new
   - Repository name: `elite`
   - Make it **Public**
   - Click **Create repository**

3. **Upload files using GitHub web interface:**
   - On the new repository page, click **"uploading an existing file"**
   - Drag and drop ALL files from your Elite folder:
     - `index.html`
     - `styles.css`
     - `app.js`
     - `product-modal.css`
     - `logo.png`
     - `bg.png`
     - `README.md`
   - Scroll down, add commit message: "Initial commit"
   - Click **"Commit changes"**

4. **Enable GitHub Pages:**
   - Go to: **Settings** (top menu)
   - Scroll to **Pages** (left sidebar)
   - Under **Source**, select:
     - Branch: `main`
     - Folder: `/ (root)`
   - Click **Save**

5. **Your site is live!**
   - URL: `https://arunmaroon.github.io/elite/`
   - (Takes 1-2 minutes to go live)
