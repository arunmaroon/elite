# Hero Banner Image Setup

## To Use Your Kitchen Image

The hero section is ready to use your kitchen image with black cast iron cookware on grey concrete wall.

### Option 1: Add Image File
1. Save your hero image as `hero-banner.jpg` (or `.png`) in the same folder as `index.html`
2. Update `styles.css` line ~430:
   ```css
   background-image: url('hero-banner.jpg');
   ```

### Option 2: Use Image URL
1. Upload your image to a hosting service or use the image URL
2. Update `styles.css` line ~430:
   ```css
   background-image: url('YOUR_IMAGE_URL_HERE');
   ```

### Option 3: Inline Style (Quick)
Add the image URL directly in `index.html`:
```html
<div class="hero__background" style="background-image: url('YOUR_IMAGE_URL');"></div>
```
